# India Wildlife Explorer 🌿🐅

A responsive wildlife tourism website built using HTML, CSS, JavaScript and Bootstrap.

---

## 📌 Project Overview

India Wildlife Explorer is a beginner-friendly frontend project that showcases famous wildlife sanctuaries and national parks in India.

The website contains multiple pages with wildlife information, images and navigation links.

---

## 🚀 Features

- Responsive Bootstrap design
- Multi-page website
- Wildlife sanctuary information
- Explore buttons with navigation
- Beautiful wildlife images
- Navbar and footer sections
- GitHub Pages deployment

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
